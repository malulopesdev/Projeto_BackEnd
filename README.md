Cadastro de Produtos Sustentáveis
O presente projeto tem como objetivo o desenvolvimento de um sistema Back-End CRUD para o gerenciamento de produtos ecológicos e sustentáveis. O foco é catalogar itens que incentivam escolhas conscientes de consumo, alinhando-se aos Objetivos de Desenvolvimento Sustentável (ODS) da ONU, em especial os voltados para consumo e produção responsáveis e a preservação ambiental. 

Tecnologias Utilizadas

Node.js & Express → Framework principal para a construção da API e rotas do servidor. 


MySQL → Banco de dados relacional para persistência das informações dos produtos. 


EJS (Embedded JavaScript templates) → Engine de visualização para renderizar as páginas HTML dinamicamente. 


CSS3 → Estilização moderna, harmônica e focada em uma experiência de usuário limpa. 

Estrutura da Aplicação
A aplicação é composta por quatro seções principais, conforme os requisitos da AV1: 


Home → Página inicial com a apresentação do projeto e identidade visual. 


Sobre → Detalhes sobre a missão do projeto e sua conexão com as metas de sustentabilidade da ONU. 


Produtos → Vitrine que lista todos os itens cadastrados no banco de dados em tempo real. 


Cadastro/Edição → Formulários dinâmicos que permitem inserir novos produtos ou atualizar dados existentes. 

Funcionalidades (CRUD)

Create (Cadastrar): Inclusão de novos produtos com campos para nome, preço, descrição, fornecedor, e-mail e telefone. 


Read (Listar): Exibição organizada de todos os produtos salvos no MySQL. 


Update (Editar): Possibilidade de alterar informações de produtos já cadastrados. 


Delete (Excluir): Remoção definitiva de itens do banco de dados através da interface.
